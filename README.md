# Loan_Default_PowerBI
Designed an interactive Power BI dashboard to analyze loan default patterns by borrower demographics, loan characteristics financial health. Performed data preparation in Power Query and created DAX measures for default rates and key metrics. Built KPI cards, bar charts, scatter plt , matrices and cross-filtering for multi-dimensional risk analysis

# Loan Default Analysis Dashboard

## Overview
Power BI dashboard analyzing 255,347 loan records 
with 11.61% default rate across 5 report pages.

## Key Metrics
- Total Loans: 255,347
- Total Loan Amount: 33bn
- Total Defaults: 29,653
- Default Rate: 11.61%

## Dashboard Pages
1. Executive Summary
2. Borrower Risk Profile
3. Loan Characteristics
4. Credit & Financial Health
5. Risk Factor Deep Dive

## Tools Used
- Power BI Desktop
- DAX (CALCULATE, DIVIDE, FILTER, COUNTROWS)
- Power Query (M Language)

## Dataset
255,347 records | 19 columns | 2024-2026
